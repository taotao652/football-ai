# 足球预测模型说明

## 📁 文件结构

### 核心模型文件

#### 1. `ensemble_predict.py` ⭐ **主力模型**
- **准确率**: 51.58%
- **方法**: 集成模型（泊松分布 + Elo评级 + Dixon-Coles修正）
- **特点**: 
  - 结合了多种预测方法的优势
  - 主胜预测准确率高达 79.35%
  - 高置信度预测（>55%）准确率达 58.29%
- **使用方法**:
  ```bash
  python ensemble_predict.py
  ```

#### 2. `poisson_predict.py` 
- **准确率**: 48.68%
- **方法**: 基础泊松分布模型
- **特点**: 
  - 简单稳定，易于理解
  - 适合作为基线模型
- **使用方法**:
  ```bash
  python poisson_predict.py
  ```

### 数据采集文件

#### 3. `fetch_enriched_data.py`
- **用途**: 从 API-FOOTBALL 获取详细的比赛统计数据
- **功能特性**:
  - 断点续传：中断后自动从上次位置继续
  - 额度检测：API 配额用完自动停止并保存进度
  - 实时保存：每成功一场立即写入磁盘
- **可获取特征**:
  - 射门（射正/射偏/总射门/被封堵/禁区内/禁区外）
  - 控球率、角球、犯规、黄牌/红牌
  - 越位、传球（总数/成功数/成功率）、抢断、扑救
- **配置**: 在项目根目录 `.env` 文件中设置 `API_FOOTBALL_KEY`
- **使用方法**:
  ```bash
  python fetch_enriched_data.py
  ```
- **注意**: 免费套餐仅支持 2022 赛季统计数据，每日约 10 次请求

#### 4. `fetch_and_save.py`
- **用途**: 原始比赛数据抓取脚本（下载基础比分和赛程数据）
- **说明**: 一次性获取整个赛季的 fixture 列表，保存为 JSON

### 其他文件

#### 5. `train_model.py`
- **用途**: 随机森林分类器训练脚本
- **方法**: 基于球队历史场均进球/失球/胜率特征
- **说明**: 使用 22+23 赛季训练，24 赛季测试

#### 6. `data/` 目录
- `premier_league_2022.json` - 2022赛季基础数据（380场）
- `premier_league_2023.json` - 2023赛季基础数据（仅比分）
- `premier_league_2024.json` - 2024赛季基础数据（仅比分）
- `enriched_2022.json` - 2022赛季增强数据（含统计特征，采集中）

## 🚀 快速开始

### 运行预测模型
```bash
# 使用最佳模型

# 或使用基础模型

```


## 💡 优化建议

### 当前瓶颈
- 仅使用比分数据，特征有限
- 平局预测仍然较弱（泊松分布的固有缺陷）

### 未来改进方向
1. **增加数据特征** (最重要)
   - 获取射门、控球率等详细统计
   - 运行 `fetch_enriched_data.py` 采集数据
   
2. **引入更多特征**
   - 球队身价、伤病信息
   - 天气、裁判等因素
   
3. **使用机器学习**
   - 有了足够特征后，XGBoost/LightGBM 会有更好表现

## 🔑 API 配置

数据采集需要在项目根目录创建 `.env` 文件：

```bash
# 在 football-ai/ 目录下
echo "API_FOOTBALL_KEY=你的密钥" > .env
```

脚本会自动加载，无需修改代码。

---

## 📝 注意事项

1. **API 额度限制**: 免费套餐每日约 10 次请求（统计数据端点），用完自动停止
2. **赛季限制**: 免费套餐仅支持 2022 赛季的统计数据，23/24 需付费
3. **数据保存**: 每成功一场立即写入 `data/enriched_*.json`，中断不丢数据
4. **断点续传**: 再次运行自动从上次中断处继续，无需手动干预
